## Overview

As a **Software Engineer** at **Dinamo**, I developed and delivered **4 high-performance, content-driven portfolio websites** for our client **Media Edge Group** — a leading Motion Graphics & VFX studio in Egypt.

All four projects were built using the **same modern tech stack** and architecture, pulling dynamic content from a central CMS/API. This allowed the client to manage all copy, images, reels, and projects easily without touching the code.

The websites showcase motion graphics artists and the studio itself with smooth animations, excellent SEO, and fast loading times.

## Shared Tech Stack & Architecture

- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript (nearly 100%)
- **Styling:** Tailwind CSS v4
- **Data Fetching:** TanStack React Query + Axios
- **Animations:** Framer Motion
- **Other:** React Slick, Remote Image Optimization, Dynamic Metadata
- **Deployment:** Vercel (optimized)

**Architecture Highlights:**
- Component-driven & reusable UI
- Clean folder structure
- Server Components + Client Components balance
- Centralized CMS integration (`content.mediaedgegroup.com`)
